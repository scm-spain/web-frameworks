Box

Establecemos una caja base que utilizaremos, en ocasiones, como parte de otros componentes.

El componente Box se monta a partir de un section con la clase .box, una cabecera h1 con la clase .box_title y el contenido dentro de un div con la clase .box_content.

CSS CLASSES: box box-title box-content

Show code 
                                       
        <section class="box">
            <h1 class="box_title">Box Title</h1>
            <div class="box_content">

                <p>Lorem ipsum dolor sit amet, consectetur adipisicing elit.</p>

        </section>

Definimos el componente Switchlist, que puede utilizarse de manera individual para establecer una lista de enlaces o como parte de otros componentes como Dropdowns.

Switchlist

El switchlist básico se monta con la clase .o-link-list aplicada al div contenedor. Lo que obtenemos es un listado vertical con los estilos definidos para los estados hover y selected.

CSS CLASSES: o-link-list selected

Show code 
                                           
    <div class="o-link-list">
        <ul>
            <li>
                <input type="radio" name="switch-name" />
                <label>This is the first item</label>
            </li>
            <li>
                <input type="radio" name="switch-name" />
                <label>This is the second item</label>
            </li>
            <li>
                <input type="radio" name="switch-name" />
                <label>Yes... this is the thrid item</label>
            </li>
            <li class="selected">
                <input type="radio" name="switch-name" />
                <label>And this is a selected item</label>
            </li>
        </ul>
    </div>

DATA REQUIRED: data-component="switchlist" id

Buttons

Podemos crear botones de dos maneras distintas: a partir de la etiqueta <button></button>, o estableciendo que un link tenga apariencia de botón. A continuación veremos la manera de llevar a cabo ambos métodos de creación de botones.

Partiremos siempre de una clase general button. Dicha clase nos ofrece un botón genérico de tamaño medio. A partir de la clase button podemos crear variaciones de botones según su tamaño (button--small, button--large) o su prioridad (button--primary, button--accent o disabled.

Dialog

Mostramos la manera de abrir una ventana modal desde un enlace. Es muy importante que el enlace (o cualquier otro elemento inductor) contenga el data-dialog="inducer" y también el data-target="x" con el "id" de la modal correspondiente.

CSS CLASSES: dialog_wrapper dialog_overlay dialog dialog_header dialog_title dialog_close dialog_body dialog_footer button button--primary

DATA REQUIRED: data-dialog="inducer" data-target id

Show code 
                            
    <!--Enlace que desencadena la apertura de la modal.-->          
    <a data-dialog="inducer" data-target="dialog-generic">This is a link that opens a dialog component</a>
    <!--Ventana modal o de diálogo-->
    <div class="dialog_wrapper" data-module="dialog">
    <div data-dialog="overlay" class="dialog-overlay"></div>
        <section id="dialog-generic" class="dialog" role="dialog">
            <header class="dialog_header">
                <h1 class="dialog_title">dialog title</h1>
                <button class="dialog_close" data-dialog="close"></button>
                </header>
            <div class="dialog_body">
            Lorem ipsum dolor sit amet, consectetur adipiscing elit fusce vel sapien elit in malesuada semper mi, id sollicitudin urna fermentum ut fusce varius nisl ac ipsum gravida vel pretium tellus.
            </div>
            <footer class="dialog_footer">
                <button class="button" data-dialog="close">cancel</button>
                <button class="button button--primary">okay</button>
